1. What is HTML and CSS?

We use both HTML and CSS to create and design a wesbite

HTML

HTML stands for "Hypertext Markup Language". HTML is a standard markup language used to create web pages. It helps users create the structure and layout of webpages, using tags and attributes. By using these tags and attributes, users can create complex and interactive web pages. It is the bases for all the elements you can add to a webpage.

CSS

CSS stands for "Cascading Style Sheets". CSS is a styling language used to create web pages. It allows users to create all the design aspects and visual effects. CSS uses a syntax of selectors and properties to define styles for HTML elements.

2. CSS Selectors

CSS selectors allow developers to define which elements on a web page should have a particular style applied to them. This can be based on their class, ID, type and more.

The Element Selector

The element selector is a way to select all the elements with a given tag name, and apply the same styles to the element with tag name.

The Descendant Selector

The descendant selector is a way to select elements that are located underneath other elements, according to the structure. This selector is multiple selectors joined together but separated by a single space.

The Class Selector

The class selector is a way to select all elements with the specified class name and apply styles to the matching elements. The class selector must start with a period (.) and then the class name.

The ID Selector

The id selector is a way to select only the element with the specified id and apply the same styles to that element. However, you must put the id tag and the tag on the page in the same way otherwise, the element will not apply and the selector will not work
