2. Data Types and Variables

Variables and Constants

A Varible in computer programming is a data value that can be changed throughout a program. A Constant is a data value that cannot be changed once defined. Constants and variables can be different data types, such as integer, double, or string. But once a constant is defined with a data type, you cannot change it. Variables can be assigned different data types throughout the program.

For example, if you want to find the area of a circle, you have to input "pi". For a constant you must put "const double pi=3.14;" so that pi will remain the same throughout the program. For a variable, it can change throughout the program.

Input and Output

Console.WriteLine() method is used to display output on the console window and next line

Console.ReadLine() method is used to read input from the user.

Console.Write() method is used to display output on the console window without moving the cursor to the next line.

Arithemitic Operators

Arithmetic operators are used to perform operations such as addition and subtraction.

  • Addition Operator(+)- Operator adds 2 operands.
  • Subtraction Operator(-)- Operator subtracts one operand from the other.
  • Multiplication Operator(*)- Operator multiples one operand from another.
  • Devision Operator(/)- Operator divides numerator by the denominator.
  • Modulo Operator(%)- Operator provides the remainder from a divison equation.
  • Increment Operator(++)- Operator increses integer value by 1.
  • Decrement Operator(--)- Operator decreases value by 1

Logical Operators

The logical operators are used to perform specific logical operations in c#. There are three logical operators: AND, OR & NOT

  1. AND Operator(&&)- If both the operands are non zero then the condition becomes true. Ex. (A && B) is false.
  2. OR Operator(||)- If any of the two operands are non zero then the condition become true. Ex. (A || B) is true.
  3. NOT Operator(!)- If the condition is true then the NOT Operator will make it false. Reverses the logical state of the operand.

Loops

C# has several types of loops, which are used to execute a block of code repeatedly until a certain condition is met. "For", "while", and "do-while" are loops used for coding in C#.

For loops- The for loop executes one or more statements multiple times as long as the loop condition is satisfied..

While loops-The while loop continuously executes one or more statements as long as the loop condition is satisfied. If the loop condition is true, the body of the for loop is executed..

Do while- The do while loop executes one or more statements repeatedly as long as the loop condition remains true. This loop is similar to the while loop, but with a key difference: the while loop checks the condition at the beginning of the loop, while the do while loop checks it at the end. .
